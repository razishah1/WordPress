== Description ==

Sticky Floating Forms WordPress plugin allows you to add CTA buttons on your website and when the user clicks on that buttons it will display contact forms. This is useful to catch the attention of your site visitor and never miss any messages or contacts from your website. This plugin is an addon for the contact form 7 plugin but also works with all major popular form plugins.

== Frequently Asked Questions ==

Our plugin is user friendly, but here you can see some frequently asked questions that can help you.

= How can I make my forms sticky? =

After installing the plugin go to plugin setting page, and there you will get option to choose your contact form, just select your form and save the setting and you are done.

= Does your plugin supports other form plugins rather than contact form 7? =

Yes, the plugin has option to add shortcode of your form, which lets you to make any forms sticky.
